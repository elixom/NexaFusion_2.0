
Task: Build an engaging, premium layout template for the NexaFusion Testimonials page.

Strict Project Guidelines:
1. Make the template in the theme.
2. Ensure all correct WordPress comment-style block tags () are fully included and structured properly.

Layout & Technical Specifications:
- Category Filter: The page must feature a Query Loop block ('wp:query') configured to dynamically fetch and display posts assigned to the "Testimonials" category taxonomy slug.
- Post Template Elements: Inside the 'wp:post-template', each testimonial card must strictly display:
  * The Post Featured Image ('wp:post-featured-image') styled cleanly as a circular avatar image.
  * The Post Content ('wp:post-content').
  * CRITICAL: Do NOT include the Post Title block ('wp:post-title') under any circumstances.
- Aesthetics & Engagement: 
  * Design a highly visually engaging 3-column masonry or grid layout for desktop that smoothly scales responsively down to mobile layouts.
  * Group individual elements inside a card container ('wp:group') with refined visual styling parameters embedded (e.g., subtle shadows, clean border radiuses, and intentional, sophisticated layout padding) to match the NexaFusion aesthetic.
  * Style the text output within the content section to read naturally as a direct quote.
